# WorldAI — 초월자 시스템 (Transcendent System)

> 최종 수정: 2026-04-05 | 상태: Draft

---

## 1. 개념

> 레벨 체계는 **표본 집단**에 적용되는 규칙이다.
> 그러나 극히 드문 이벤트를 통해 개인이 **종족 한계를 초월**할 수 있다.
> 이들을 **초월자(Transcendent)** 라고 부른다.
>
> 초월자는 단순히 강한 것이 아니라 **존재 자체가 바뀌는 것**이다.
> 드래곤 하트를 먹은 인간은 인간이면서 드래곤의 일부가 된다.

---

## 2. 초월 조건 (Trigger Events)

초월은 **희소 이벤트**로만 발생한다. 무작위 발생이 아니라 특정 조건을 충족해야 한다.

| 초월 유형 | 트리거 조건 | 결과 |
|-----------|------------|------|
| **용인 (Dragonborn)** | 드래곤을 단독 처치 + 드래곤 하트 섭취 | 인간→용인 진화. 드래곤 능력 일부 획득 |
| **신의 사자** | 천사/악마에게 직접 선택받음 | 해당 진영 능력 도약. 수명 연장 |
| **불사 흡혈귀** | 뱀파이어에게 혈통 이식, 의지로 수용 | 인간→뱀파이어 고급 계열 |
| **원소 합일** | 대정령과 완전 융합 | 해당 원소 지형에서 정령 능력 획득 |
| **마계 계약자** | 마왕급 악마와 계약 | 악마 능력 일부. 영혼 일부 대가 |
| **고대 문자 해독** | 고대 드래곤 문자 전부 습득 | 마법 한계 돌파. 특수 마법 해금 |
| **신물 보유자** | 세계 5대 신물 중 하나 획득 | 신물에 따른 특수 능력 |

---

## 3. 초월 단계

초월자도 단계가 있다. 단번에 최고가 되지 않는다.

### 용인 (Dragonborn) 예시

```
인간 전사 (레벨 65)
    ↓ [드래곤 토벌 + 하트 섭취 이벤트]
초월 단계 1: 용인 (半용인)
  - 레벨 상한: 인간 99 → 250으로 확장
  - 드래곤 불꽃 약화 버전 획득
  - 드래곤과 대화 가능 (언어 이해)
  - 드래곤 종족이 적대에서 경계로 수정
    ↓ [추가 레벨 업 + 드래곤 소울 2개 흡수]
초월 단계 2: 고위 용인
  - 레벨 상한: 350으로 확장
  - 비행 능력 획득 (날개 성장)
  - 드래곤 마법 사용 가능
    ↓ [극히 드문 추가 이벤트]
초월 단계 3: 용의 화신 (Dragon Avatar)
  - 레벨 상한: 600
  - 용왕에게도 인정받는 존재
  - 드래곤 종족 일부 지휘 가능
```

## 4. 초월자의 세계 영향

초월자는 시뮬레이션에서 **특별한 존재**로 취급된다.

### 영향 범위

| 효과 | 설명 |
|------|------|
| **이름 있는 존재** | 전 종족에 존재가 알려짐. 이름이 역사 기록에 등재 |
| **파벌 버프** | 소속 파벌의 사기(morale) +20%, 모집 인구 증가 |
| **외교 영향** | 초월 유형에 관련된 종족과 자동 외교 변화 |
| **이벤트 트리거** | 초월자 존재 자체가 세계 이벤트를 유발할 수 있음 |
| **표적** | 반대 진영이 우선 제거 대상으로 삼음 |

### 예시 이벤트

```yaml
# 용인 탄생 시 자동 발생 이벤트들
triggered_events:
  - type: WORLD_ANNOUNCEMENT
    title: "용인이 나타났다!"
    description: "드래곤을 쓰러뜨린 영웅이 용의 힘을 얻었다."
    affected_all: true

  - type: DRAGON_REACTION
    condition: "dragon_exists"
    description: "드래곤들이 용인의 존재를 감지. 일부는 적대, 일부는 시험"
    affinity_change:
      dragon → transcendent_owner_faction: -20 to +10 (random)

  - type: RELIGION_SURGE
    condition: "dragon_cult_exists"
    description: "드래곤 교단이 용인을 신의 사자로 선포"
    effect: dragon_cult_spread +30%
```

## 5. 초월 YAML 스키마

```yaml
# 캐릭터 데이터에 추가되는 초월 정보
character:
  id: hero_aldric
  name: "알드릭 드래곤베인"
  race: human
  base_level: 65          # 초월 전 원래 레벨
  
  transcendent: true
  transcendent_type: dragonborn
  transcendent_stage: 1   # 1, 2, 3
  transcendent_source: "북방 설봉의 노룡 그리말 처치"
  
  # 초월로 추가된 능력
  transcendent_traits:
    - dragon_breath_weak   # 드래곤 불꽃 약화 버전
    - dragon_tongue        # 드래곤 언어 이해
    - scaled_skin          # 물리 내성 +10%
    - soul_resonance       # 드래곤 영혼 공명 (레벨 성장 가속)
  
  # 초월 후 레벨 상한
  level_cap: 250           # 원래 99 → 250
  current_level: 78        # 초월 후 레벨 재산정
  
  # 외교 영향
  auto_affinity_changes:
    - target_race: dragon
      delta: -10           # 드래곤이 경계함 (+관계 변화 랜덤)
    - target_religion: dragon_cult
      delta: +35           # 드래곤 교단이 열렬 환영
```

## 6. 세계 5대 신물 (World Relics)

신물 보유만으로도 초월에 준하는 능력을 얻는다. 신물은 세계에 딱 5개뿐이다.

| 신물 | 설명 | 효과 |
|------|------|------|
| **용왕의 심장** | 전설의 용왕 잔해 | 보유자 레벨 상한 +200, 드래곤 종족 지휘 가능 |
| **천계의 날개** | 대천사의 깃털로 만든 유물 | 보유 파벌 언데드/악마 전투 +40%, 치유 능력 |
| **마계의 계약서** | 대마왕이 서명한 고대 계약 | 악마 소환 가능, 마왕급과 협상 가능 |
| **세계수의 씨앗** | 대정령이 봉인한 원소 에너지 | 어느 지형에서나 정령 능력 발현 |
| **불사의 돌** | 리치가 만든 철학의 돌 유사품 | 보유자 사망 시 1회 부활. 언데드 저항 |

### 신물 획득 조건
- 고대 유적 탐험 (확률 1/1000)
- 해당 신물과 연관된 존재 처치 (용왕 처치 → 용왕의 심장)
- 비밀 거래 (마왕급과 외교)

## 7. 초월과 게임 밸런스

초월자는 강하지만 **수명 대가**가 있다.

```yaml
transcendent_costs:
  dragonborn:
    - 인간 공동체에서 점차 이질감. 인간 파벌 morale 영향
    - 드래곤 교단에 의존도 증가
    - 수명 연장되나 고독
  
  divine_chosen:
    - 선택한 진영(천사/악마)의 반대 진영에게 최우선 표적
    - 자유로운 중립 외교 불가
  
  demon_contractor:
    - 계약 조건 이행 실패 시 영혼 박탈 이벤트
    - 천사/빛 종교 파벌과 외교 불가(-80 고정)
```
