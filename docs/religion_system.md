# WorldAI — 종교 시스템 (Religion System)

> 최종 수정: 2026-04-05 | 상태: Draft

---

## 1. 설계 원칙

> 종교는 특정 초월 존재(드래곤, 천사, 악마, 정령)를 숭배하는 **교단** 형태로 존재한다.
> 교단은 특정 파벌에 종속되지 않고 **어느 마을에나 교세를 뻗을 수 있다.**
> 교단들은 서로 적대적일 수 있으며, 같은 파벌 안에 여러 교단이 공존·충돌할 수 있다.

---

## 2. 숭배 대상 유형 (Deity Type)

| 유형 ID | 숭배 대상 | 주요 교세 지역 | 특징 |
|---------|-----------|---------------|------|
| `dragon_cult` | 드래곤 (특정 개체 or 드래곤족 전체) | 산악, 용 서식 인근 | 힘과 지혜 숭배. 드래곤에게 제물 바침 |
| `angel_faith` | 천사 / 천신 | 중앙 평원, 번성한 도시 | 빛과 질서. 적대 종교 정화 추구 |
| `demon_cult` | 악마 / 마계 | 황무지, 하층민 밀집 지역 | 금지된 힘 추구. 지하 활동 |
| `nature_faith` | 대정령 / 자연 그 자체 | 숲, 설원, 수인 지역 | 엘프·수인 선호. 자연 보전 |
| `ancestor_faith` | 조상신 | 드워프, 오크 | 선조와의 연결. 가문·부족 중시 |
| `undead_cult` | 언데드 군주 / 리치 | 황무지 인근, 공포 지역 | 불사 추구. 사회 금기 |

## 3. 교단 구조 (Church Structure)

### 규모 단계

| 규모 | 교세 | 구성원 | 설명 |
|------|------|--------|------|
| 소수 신자 | 0 ~ 15% | 개인·가문 | 공식 조직 없음. 비밀 신앙 |
| 소교단 | 16 ~ 35% | 마을 단위 | 교주·사제 존재. 공개 활동 |
| 중교단 | 36 ~ 60% | 도시 단위 | 건물(성당/신전/제단) 보유 |
| 대교단 | 61 ~ 80% | 왕국 단위 | 국교 수준. 파벌 정치에 개입 |
| 국교 / 지배 교단 | 81 ~ 100% | 제국 단위 | 지도자가 교단 수장 겸임 가능 |

### 교단 영향권

```yaml
faction_religion:
  dominant: angel_faith     # 지배 종교 (61%)
  minority:
    - demon_cult: 15%       # 하층민 사이 암약
    - nature_faith: 10%     # 숲 인근 농민
    - none: 14%             # 무신론자
```

## 4. 교단 상호 관계

### 기본 갈등 구도

```
  천사 교단   ←→ 적대 ←→   악마 교단
  자연 신앙   ←→ 적대 ←→   언데드 교단
  드래곤 교단 ←→ 긴장 ←→   천사 교단 (천사는 드래곤을 신격화 거부)
  조상 신앙   ←→ 중립 ←→   대부분 (타 신앙 간섭 없음)
```

### 친밀도 기본값 (종교 간)

| 종교 A | 종교 B | 기본 친밀도 | 이유 |
|--------|--------|------------|------|
| 천사 신앙 | 악마 교단 | -90 | 본질적 대립 |
| 천사 신앙 | 자연 신앙 | -20 | 신격화 방식 불일치 |
| 천사 신앙 | 드래곤 교단 | -40 | 드래곤을 신으로 인정 안 함 |
| 자연 신앙 | 언데드 교단 | -80 | 생명과 죽음의 대립 |
| 드래곤 교단 | 조상 신앙 | +10 | 힘 숭배 공통점 |
| 악마 교단 | 언데드 교단 | +30 | 어둠 계열 연대 |

## 5. 종교 이벤트

### 정기 이벤트

| 이벤트 | 발생 조건 | 효과 |
|--------|-----------|------|
| 종교 전파 | 매 168틱 (7일) | 인접 파벌에 교세 +5% |
| 종교 박해 | 지배 교단이 타 교단 탐지 | 소수 교단 세력 -20%, 갈등 이벤트 |
| 십자군 / 성전 | 교세가 70% 이상이고 적대 교단 인접 | 전쟁 이벤트 연결 가능 |
| 사제 파견 | 교단 규모 중교단 이상 | 타 파벌에 교사 파견 → 교세 전파 |
| 기적 발생 | 1%/년 | 승배 대상(드래곤 등)이 신자를 직접 돕는 이벤트 → 교세 폭발 |

### 능력치 영향

| 지배 종교 | 파벌 효과 |
|-----------|-----------|
| 천사 신앙 | morale +10%, 언데드/악마와 전투 시 +15% |
| 악마 교단 | 공격력 +10%, 아군 사기 -5%（불안감） |
| 자연 신앙 | 식량 생산 +15%, 마법 효율 +10% |
| 드래곤 교단 | 드래곤과 친밀도 +20 자동, 기술 연구 +5% |
| 조상 신앙 | 방어력 +10%, 영토 이탈 시 사기 -20% |
| 언데드 교단 | 공포 효과 +10%, 생명 종족과 외교 -15% |

## 6. YAML 스키마

```yaml
# configs/religions/angel_faith.yaml
id: angel_faith
name: "천계의 빛 교단"
deity_type: angel
deity_id: angel_demon       # 연결된 종족 또는 신격 ID
description: "빛과 질서를 추구하는 신앙. 어둠과 혼돈을 정화하려 한다."

base_affinity:              # 타 종교와 기본 친밀도
  demon_cult: -90
  undead_cult: -70
  dragon_cult: -40
  nature_faith: -20
  ancestor_faith: 0

spread_rate: 0.05           # 틱당 인접 파벌 교세 전파율
conflict_threshold: 30      # 타 교단이 이 수치 이상 교세 가지면 충돌 이벤트
tension_rate: 0.02          # 매 틱 갈등 수치 상승

faction_effects:
  morale: +0.10
  combat_vs_undead: +0.15
  combat_vs_demon: +0.15
  diplomacy_with_fairy: +0.10
```

## 7. 종교와 파벌 정치

### 국교 선언 조건
- 지배 교단 교세가 80% 이상
- 지도자가 해당 교단의 신자
- 선언 시: 교단 효과 강화, 타 종교 박해 시작, 특정 종족과 자동 외교 보정

### 종교 분쟁 → 내전 연결
- 같은 파벌 내 두 교단 교세 합이 90%이상이고 두 교단이 적대면 내전 이벤트 가능
- 내전 발생 시 파벌이 두 개의 새 파벌로 분열
