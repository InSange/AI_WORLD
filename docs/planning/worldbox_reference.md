## 3. 문명 시스템

### WorldBox 문명 메카닉
- **마을 성장**: 작은 마을 → 도시 → 왕국으로 자동 성장
- **왕국 확장**: 인접 빈 타일 점령, 다른 왕국과 영토 충돌
- **전쟁 선포**: 영토 충돌 or 자원 부족 시 자동 전쟁
- **평화 협정**: 전쟁 후 피해가 크면 평화 요청
- **동맹**: 왕국 간 동맹 체결, 동맹국 전쟁 시 자동 참전
- **반란**: 왕국 내 불만 누적 → 독립 반란 발생
- **클랜 시스템**: 왕/지도자들이 클랜을 형성, 혈통 계승
- **시대(Ages)**: 세계가 시대를 거치며 변화 (선사시대 → 중세 → 근대 등)
- **배너/문장**: 왕국 고유 문양 커스터마이징

### WorldAI 현황 대비
| WorldBox | WorldAI | 갭 |
|----------|---------|-----|
| 마을→왕국 자동 성장 | ✅ FactionManager 규모 체계 | 있음 |
| 영토 점령/충돌 | ✅ DirtyRegion 기반 | 있음 |
| 전쟁 선포 | ✅ 외교 임계값 이벤트 | 있음 |
| 동맹 체결 | ✅ ALLIED 친밀도 단계 | 있음 |
| 반란/독립 | ❌ 미구현 | **HIGH 우선순위** |
| 클랜/혈통 계승 | ❌ 미구현 | **MEDIUM 우선순위** |
| 세계 시대 (Ages) | ❌ 미구현 | **HIGH 우선순위** |
| 배너/문장 | ❌ 미구현 | LOW (비주얼) |
| 해상 항로/섬 탐험 | ❌ 미구현 | **MEDIUM 우선순위** |

## 4. 신의 능력 / 재앙

### WorldBox 신의 능력 카테고리
| 카테고리 | 능력 목록 |
|----------|-----------|
| **생명 창조** | 종족 소환, 동물 소환, 씨앗 심기 |
| **지형 변환** | 지형 타입 변경, 물 추가/제거, 산 생성 |
| **축복** | 특정 마을 성장 가속, 왕국 강화, 자원 부여 |
| **재앙** | 번개, 토네이도, 지진, 화산 폭발, 산성비 |
| **역병** | 전염병 살포 (좀비 역병, 흑사병 등) |
| **파괴** | 폭발, 핵폭탄, 유성 낙하, 블랙홀 |
| **날씨 조작** | 비, 눈, 가뭄 |
| **저주** | 특정 왕국 약화, 지도자 암살 |

### WorldAI 현황 대비
| WorldBox | WorldAI | 갭 |
|----------|---------|-----|
| 번개/지진/화산 | ❌ 없음 | **추가 검토** |
| 전염병 | ✅ 역병 이벤트 있음 | 있음 |
| 축복/저주 | ❌ 직접 개입 없음 | **MEDIUM** |
| 유성/핵폭탄 | ❌ 없음 | LOW |
| 날씨 조작 | ❌ 없음 | **MEDIUM** |
| 지형 실시간 변환 | ❌ 없음 | **MEDIUM** |
| **플레이어가 신** | ❌ 관찰자 전용 | **핵심 방향성 결정 필요** |

> ⚠️ **중요 설계 결정**: WorldAI는 현재 "자율 시뮬레이션 관찰" 구조다.  
> WorldBox처럼 "플레이어가 신이 되어 개입"하는 구조를 추가할지 여부가  
> 아키텍처 방향을 크게 바꾼다. **플레이어 그리드 API**(백로그 HIGH)와 연결됨.

## 5. 트레잇 (Traits) 시스템

### WorldBox 트레잇 종류
- **유닛 개인 트레잇**: 용감함, 겁쟁이, 강함, 약함, 빠름, 느림, 현명함, 어리석음
- **왕국 트레잇**: 호전적, 평화적, 종교적, 상업적, 고립적
- **전파**: 부모→자식 일부 유전, 경험으로 새 트레잇 획득

### WorldAI 현황 대비
| WorldBox | WorldAI | 갭 |
|----------|---------|-----|
| 왕국 성향 트레잇 | ✅ behavior(aggression 등) YAML | 있음 |
| 유닛 개인 트레잇 | 초월자 시스템으로 일부 ✅ | 부분 구현 |
| 트레잇 유전/전파 | ❌ 없음 | **MEDIUM** |
| 경험으로 트레잇 변화 | ❌ 없음 | **MEDIUM** |

## 6. 기술 연구 트리

### WorldBox 기술 진보
- 문명이 자동으로 기술을 연구함 (직접 제어 없음)
- **시대별 기술**: 석기 → 청동기 → 철기 → 중세
- 기술 수준에 따라 무기/방어구 등급 변화
- 고급 기술 보유 왕국이 전투 우위

### WorldAI 현황 대비
| WorldBox | WorldAI | 갭 |
|----------|---------|-----|
| 자동 기술 연구 | ✅ technology_level 스탯 있음 | 있음 |
| 시대별 기술 단계 | ❌ 단일 수치만 존재 | **MEDIUM** |
| 기술 수준별 유닛 외형 변화 | ❌ 없음 | LOW |

## 7. 종교 / 신앙 시스템

### WorldBox 종교
- 왕국이 특정 신을 숭배
- 같은 신을 믿는 왕국끼리 우호 경향
- 종교 전쟁 발생 가능
- 신앙심이 높으면 기적 발생 확률 증가

### WorldAI 현황 대비
| WorldBox | WorldAI | 갭 |
|----------|---------|-----|
| 왕국별 종교 | ✅ religion_system.md 구현 완료 | 있음 |
| 종교 전쟁 | ✅ 교단 갈등 이벤트 | 있음 |
| 기적 이벤트 | ✅ 1%/년 기적 발생 | 있음 |

## 8. 세계 시대 (Ages) 시스템

### WorldBox Ages
- 세계가 일정 조건 충족 시 다음 시대로 전환
- 시대 전환 시 바이옴/생물 변화
- 빙하기, 화산 폭발기, 황금기 등
- 시대별 새 종족/기술 언락

### WorldAI 현황 대비
| WorldBox | WorldAI | 갭 |
|----------|---------|-----|
| 세계 시대 진행 | ❌ 없음 | **HIGH 우선순위** |
| 시대별 이벤트 | ❌ 없음 | **HIGH 우선순위** |
| 시대 조건 (연도/틱) | ❌ 없음 | **HIGH 우선순위** |

| 항목 | WorldBox | WorldAI |

| 파벌 계층 | 왕국 단위 | **전초기지~제국 6단계** (WorldAI 우세) |
| 시대 시스템 | ✅ 있음 | ❌ **WorldAI 미구현** |

| 바이옴별 자원 | ✅ 있음 | ❌ **WorldAI 미구현** |
