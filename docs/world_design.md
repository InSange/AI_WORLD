# WorldAI — 세계관 기획서 (World Design)

> 최종 수정: 2026-04-05 | 상태: Draft

---

## 1. 세계 개요

### 세계 이름
**아스테리아 (Asteria)** — "별들의 땅"이라는 뜻의 고대어. 기본 세계관이며 사용자가 YAML로 교체 가능.

### 세계 규모
- 단일 대륙: **에테르 대륙 (Aether Continent)**
- 크기: 가로 100 × 세로 80 (타일 기준)
- 지형: 다양한 바이옴이 공존하는 복합 대륙

---

## 2. 시간 시스템

### 기본 단위

| 단위 | 정의 | 설명 |
|------|------|------|
| **틱 (Tick)** | 시뮬레이션 최소 단위 | 약 1일에 해당 |
| **계절 (Season)** | 90틱 | 봄·여름·가을·겨울 |
| **연도 (Year)** | 360틱 (4계절) | 1년 |

### 계절 효과

| 계절 | 효과 |
|------|------|
| 🌱 **봄 (Spring)** | 인구 성장 +10%, 농업 생산 +20%, 이주 활발 |
| ☀️ **여름 (Summer)** | 전투력 +5%, 탐험 활발, 교역 최대 |
| 🍂 **가을 (Autumn)** | 식량 비축, 외교 협상 선호, 전쟁 준비 |
| ❄️ **겨울 (Winter)** | 인구 성장 -20%, 전투 불리, 실내 기술 연구 +15% |

---

## 3. 대륙 지형 (에테르 대륙)

### 지형 종류 및 특성

| 지형 | 기호 | 이동 비용 | 방어 보정 | 특산 자원 |
|------|------|---------|---------|---------|
| 평원 | 🟩 | 1 | 0% | 식량, 말 |
| 숲 | 🌲 | 2 | +20% | 목재, 약초, 엘프 서식지 |
| 산악 | ⛰️ | 3 | +40% | 광물, 드워프 서식지 |
| 사막 | 🏜️ | 2 | -10% | 희귀 광물, 유리 |
| 설원 | ❄️ | 3 | +10% | 모피, 얼음 |
| 강/해안 | 🌊 | 2 | -20% | 어류, 교역 요충지 |
| 마법 지대 | ✨ | 2 | +30% | 마정석, 마법 자원 |
| 황무지 | 💀 | 1 | -20% | 뼈, 광석 (언데드 서식지) |

### 대륙 지역 구분

```
[설원]──────[마법 지대]──────[산악]
  │              │              │
[숲]────[평원 (중앙)]────[사막]
  │              │              │
[해안]──────[강유역]──────[황무지]
```

**중앙 평원**: 인간 문명의 중심지. 가장 비옥하고 전략적 요충지.
**북부 설원**: 수인족, 드래곤 서식지. 혹독한 환경.
**동부 산악**: 드워프 왕국. 광맥 밀집 지역.
**서부 숲**: 엘프 문명. 고대 마법의 흔적.
**남부 황무지**: 언데드, 악마 출몰 지역.
**마법 지대**: 어디에나 산재. 정령·골렘·드래곤이 거주.

---

## 4. 자원 시스템

### 자원 종류

| 분류 | 자원 | 획득 지형 | 용도 |
|------|------|---------|------|
| **식량** | 곡물, 어류, 사냥감 | 평원, 강, 숲 | 인구 유지 |
| **광물** | 철, 금, 미스릴 | 산악, 사막 | 무기/건설 |
| **마법** | 마정석, 허브, 영약 재료 | 마법 지대, 숲 | 마법 연구 |
| **목재** | 일반 목재, 고대목 | 숲 | 건설, 함선 |
| **희귀** | 드래곤 비늘, 언데드 뼈 | 황무지, 산악 | 특수 제작 |

## 5. 세계 이벤트

### 자연 이벤트 (랜덤 발생)

| 이벤트 | 확률 | 효과 |
|--------|------|------|
| 대풍년 | 5%/년 | 전 종족 식량 +50% |
| 대기근 | 3%/년 | 전 종족 인구 -15% |
| 마법 폭풍 | 2%/년 | 마법 지대 확장, 마법 종족 강화 |
| 역병 | 4%/년 | 특정 지역 인구 -30% |
| 고대 유적 발견 | 3%/년 | 발견 종족 기술 +20% |
| 드래곤 각성 | 1%/년 | 드래곤 세력 급성장, 주변 종족 위협 |

## 6. YAML 설정 예시

```yaml
# configs/worlds/asteria.yaml
id: asteria
name: 아스테리아 (Asteria)
description: "별들의 땅. 다양한 종족이 공존하는 판타지 대륙."

map:
  width: 100
  height: 80
  name: 에테르 대륙

time:
  ticks_per_season: 90
  seasons: [spring, summer, autumn, winter]

starting_races:
  - id: human
    start_region: central_plains
    population: 2000
  - id: elf
    start_region: western_forest
    population: 800
  - id: dwarf
    start_region: eastern_mountain
    population: 600

world_events:
  enabled: true
  base_rate: 0.03  # 틱당 이벤트 발생 기본 확률
```
