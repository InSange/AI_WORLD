# WorldAI — 등급·레벨 시스템 (Rank & Level System)

> 최종 수정: 2026-04-05 | 상태: Draft

---

## 1. 핵심 개념 구분

이 시스템에서 **등급**, **레벨**, **직위**는 서로 다른 축이다.

| 축 | 정의 | 예시 | 비고 |
|----|------|------|------|
| **등급 (Grade)** | 종족 내 개인의 강함 분류 | 하급악마, 중급악마, 소드마스터 | 능력치 기반 분류 |
| **레벨 (Level)** | 개인 능력의 수치화 (1 ~ 999) | 레벨 45, 레벨 300 | 장수 + 경험으로 증가 |
| **직위 (Title)** | 정치·사회적 위계 | 왕, 황제, 촌장, 대마왕 | 등급·레벨과 무관할 수 있음 |

### 핵심 규칙

```
✅ 레벨이 높으면 등급이 높다.
✅ 등급이 높으면 강하다.
❌ 직위가 높다고 레벨이 높은 것은 아니다.
❌ 왕이 소드마스터보다 항상 강한 것은 아니다.
✅ 단, 직위는 레벨 기준이 아니라 파벌 내 정치·혈통·무력으로 결정된다.
```

### 예외 케이스: 인간 왕 vs 소드마스터

```yaml
# 아스테리아 중앙 제국 황제
kaiser_alexan:
  name: "알렉산 3세"
  title: 황제        # 직위: 최고
  grade: 이름_있는_전사  # 등급: 중간
  level: 44          # 레벨: 낮음
  note: "혈통으로 왕위 계승. 전략가이나 개인 전투력은 평범"

# 같은 제국의 소드마스터
sword_master_veron:
  name: "베론 스틸블레이드"
  title: 제국 검사장    # 직위: 낮음
  grade: 소드마스터     # 등급: 최고
  level: 87            # 레벨: 매우 높음
  note: "황제보다 훨씬 강하지만 왕위는 없음"

# 능력 중심 파벌 -- 직위 = 최강자가 가짐
warrior_tribe_chief:
  name: "아이언피스트"
  title: 오크로드      # 직위: 최고
  grade: 전쟁추장      # 등급: 최고
  level: 78            # 레벨: 높음
  note: "오크 부족은 최강자가 우두머리. 여기선 직위=등급=일치"
```

## 2. 레벨 시스템 (Level)

### 레벨 개요

- **범위**: 1 ~ 999
- **의미**: 개인이 살아온 시간 + 겪은 전투·경험의 총합
- **성장**: 자연 성장(수명에 따른 베이스) + 경험 성장(전투·탐험·학습)
- **상한**: 99 (일반 필멸자), 200 (영생 가능 종족), 999 (신격 존재)

### 레벨 성장 공식

```python
# 자연 성장 (수명 기반)
natural_growth = current_age / max_lifespan * base_max_level

# 경험 성장 (이벤트 기반)
experience_sources = {
    "전투 승리": 0.5 ~ 3.0 레벨,   # 상대 레벨에 비례
    "마법 수련": 0.2 ~ 1.0 레벨/season,
    "탐험": 0.1 ~ 2.0 레벨,        # 고대 유적, 위험 지형
    "교역": 0.05 ~ 0.3 레벨/season,
    "수련": 0.1 ~ 0.5 레벨/season,
}

# 레벨 상한 (종족별)
level_cap = {
    "human":      99,   # 수명 짧아 성장 한계
    "elf":       350,   # 3000년 수명
    "dwarf":     200,   # 400년
    "orc":        60,   # 50년 수명
    "dragon":    999,   # 1만년 수명, 신격에 가까움
    "undead":    500,   # 불멸이나 전투력 정체
    "demon":     999,   # 마신 등급
    "angel":     999,   # 천사장 등급
    "elemental": 400,   # 재탄생 가능
    "golem":     150,   # 업그레이드로 보완
}
```

## 3. 등급 체계 (Grade System)

### 🧑 인간 (Human)

| 등급 | 레벨 범위 | 설명 |
|------|-----------|------|
| 평민 | 1 ~ 5 | 훈련 없음. 생존 능력만. |
| 병사 | 6 ~ 15 | 기초 훈련 완료 |
| 정예 병사 | 16 ~ 25 | 실전 경험 보유 |
| 기사 | 26 ~ 40 | 전문 전투 교육. 말·갑옷 운용 |
| 이름 있는 전사 | 41 ~ 55 | 전장에서 이름을 얻은 영웅 |
| 소드마스터 / 마법왕 | 56 ~ 80 | 한 분야 극한 숙달. 대륙급 강자 |
| 전설 | 81 ~ 99 | 인간 최고 한계. 설화에 남는 존재 |

> **직위 규칙**: 촌장은 최소 병사 이상. 왕은 등급 무관(혈통·정치). 소드마스터가 왕이 될 수 있음.

### 🧝 엘프 (Elf)

| 등급 | 레벨 범위 | 설명 |
|------|-----------|------|
| 새싹 | 1 ~ 20 | 초보 엘프 (수십 년) |
| 수호자 | 21 ~ 60 | 숲과 연결된 전사 |
| 현자 | 61 ~ 120 | 수백 년의 지식 축적 |
| 고위 마법사 | 121 ~ 200 | 고급 마법 운용 |
| 고대의 현자 | 201 ~ 280 | 수천 년 생존. 숲의 의지를 담당 |
| 숲의 화신 | 281 ~ 350 | 자연 그 자체와 연결. 극히 드묾 |

### ⛏️ 드워프 (Dwarf)

| 등급 | 레벨 범위 | 설명 |
|------|-----------|------|
| 견습 장인 | 1 ~ 20 | 도제 시작 |
| 장인 | 21 ~ 60 | 독립 작업 가능 |
| 명장 | 61 ~ 120 | 명성 있는 작품 제작 |
| 마스터 장인 | 121 ~ 170 | 전설급 아이템 가능 |
| 전설의 장인 | 171 ~ 200 | 드워프 역사에 이름이 남음 |

> **드워프의 전투력은 장인 등급과 비례**: 마스터 장인은 좋은 무구를 가져 전투력도 높음.

### 🪖 오크 (Orc)

| 등급 | 레벨 범위 | 설명 |
|------|-----------|------|
| 잡졸 | 1 ~ 8 | 집단에서 약자 |
| 전사 | 9 ~ 20 | 기본 전투력 보유 |
| 광전사 | 21 ~ 35 | 분노 상태 특성 발현 |
| 전쟁 추장 | 36 ~ 50 | 부족장. 집단 지휘 가능 |
| 오크로드 | 51 ~ 60 | 오크 최강. 대부족 통솔 |

> **수명 60년으로 레벨 60이 상한.** 빠른 성장이 특징이나 극한은 낮음.

### 🐉 드래곤 (Dragon)

| 등급 | 레벨 범위 | 수명 기준 | 설명 |
|------|-----------|-----------|------|
| 와이번 (아종) | 1 ~ 50 | 수십 년 | 드래곤과 구분. 비행 능력은 있으나 지성 낮음 |
| 하급용 | 51 ~ 150 | 수백 년 | 드래곤 성체. 소지역 위협 |
| 중급용 | 151 ~ 350 | 수천 년 | 광역 위협. 한 왕국 상대 가능 |
| 상급용 | 351 ~ 600 | 수만 년 | 제국도 두려워함. 고룡 |
| 고룡 (Ancient) | 601 ~ 800 | 1만 년 이상 | 전설급. 세계의 기억 보유 |
| 용왕 | 801 ~ 999 | 수만 년+ | 드래곤 중 최강. 대륙 패권급 |

> **용왕이 나타나면 전 종족에 경보 이벤트 발생.**

### 💀 언데드 (Undead)

| 등급 | 레벨 범위 | 설명 |
|------|-----------|------|
| 갓 깨어난 사체 | 1 ~ 10 | 좀비 수준. 지성 없음 |
| 해골 병사 | 11 ~ 25 | 마법으로 재편성된 전투 언데드 |
| 유령 | 26 ~ 50 | 감정·기억 보유. 지성형 |
| 뱀파이어 | 51 ~ 120 | 지성 최고. 인간사회 잠입 가능 |
| 리치 | 121 ~ 250 | 강력한 사악한 마법사. 철학적 존재 |
| 뱀파이어 군주 | 251 ~ 350 | 수백 년 뱀파이어의 정점 |
| 죽음의 군주 | 351 ~ 500 | 언데드 군단 총지휘. 대륙급 위협 |

### 😈 악마 (Demon)

| 등급 | 레벨 범위 | 설명 |
|------|-----------|------|
| 하급 악마 | 1 ~ 80 | 마계에서 가장 흔한 잡귀 |
| 중급 악마 | 81 ~ 200 | 전투 특화. 인간 세계 침투 가능 |
| 상급 악마 | 201 ~ 400 | 고급 마법 보유. 군단 지휘 |
| 대악마 | 401 ~ 600 | 마계 영지 보유. 전략가 |
| 마왕 | 601 ~ 750 | 악마 세력의 왕. 대륙 위협 |
| 대마왕 | 751 ~ 900 | 수십만 악마의 지배자 |
| 마신 | 901 ~ 999 | 신격 존재. 세계 멸망급 위협 |

> **마신은 시뮬레이션에서 직접 행동하지 않는다. 대마왕을 통해 간접 영향력만 행사.**

### 👼 천사 (Angel)

| 등급 | 레벨 범위 | 설명 |
|------|-----------|------|
| 하급 천사 | 1 ~ 80 | 신의 심부름꾼 |
| 중급 천사 | 81 ~ 200 | 수호 천사 |
| 천사장 | 201 ~ 450 | 지역 수호. 악마 토벌 |
| 대천사 | 451 ~ 700 | 천계 군단 지휘관 |
| 주천사 | 701 ~ 900 | 신의 직접 사자 |
| 천계의 화신 | 901 ~ 999 | 신과 동급. 마신과의 직접 대결 |

### 🔥 정령 (Elemental)

| 등급 | 레벨 범위 | 설명 |
|------|-----------|------|
| 소정령 | 1 ~ 30 | 약한 원소 에너지 |
| 정령 | 31 ~ 100 | 안정된 원소형 |
| 대정령 | 101 ~ 250 | 지역 원소 지배 |
| 원소의 군주 | 251 ~ 400 | 광역 원소 지배. 자연재해 유발 |

## 4. 직위(Title)와 등급의 분리

### 직위 결정 방식

파벌에 따라 지도자 선출 방식이 다르다.

| 파벌 성향 | 직위 결정 방식 | 예시 |
|-----------|----------------|------|
| 혈통 중심 | 혈연 계승 | 인간 왕국 대부분. 왕의 아들이 다음 왕 |
| 능력 중심 | 최강자가 수장 | 오크 부족, 일부 인간 전사 집단 |
| 장로 중심 | 연장자/지식 | 엘프 장로 평의회 |
| 선출 중심 | 시민 투표 | 하플링 공동체, 일부 인간 도시 |
| 마법 지배 | 강한 마법사 | 언데드 리치, 악마 마왕 |

### 직위 ↔ 등급 불일치 예시

```yaml
examples_of_mismatch:
  - case: "혈통 왕"
    character: "알렉산 3세"
    title: 황제        # 직위 최고
    grade: 이름_있는_전사  # 등급 중간 (레벨 44)
    reason: "부왕의 갑작스런 죽음으로 20세에 즉위. 전쟁보다 외교 특화"

  - case: "소드마스터 왕"
    character: "베론 스틸블레이드 2세"
    title: 왕          # 직위: 왕
    grade: 소드마스터  # 등급: 최고 (레벨 87)
    reason: "평민 출신이나 전쟁에서 공을 세워 마을의 영주가 됨 → 왕국 확장"

  - case: "약한 마왕"
    character: "네크로스"
    title: 리치 군주   # 직위: 높음
    grade: 리치        # 등급: 높음이나 최고는 아님 (레벨 180)
    reason: "강한 마법사가 아니라 뛰어난 전략가. 죽음의 군주들을 설득으로 통솔"

  - case: "오크 우두머리 = 최강자"
    character: "크롬 아이언피스트"
    title: 오크로드    # 직위: 최고
    grade: 오크로드    # 등급: 최고 (레벨 58)
    reason: "오크 부족은 순수 전투력으로만 서열 결정. 직위=등급 일치"
```

## 5. YAML 스키마 (개인 속성)

```yaml
# 개인 캐릭터 데이터 (지도자, 영웅 등)
character:
  id: veron_steelblade
  name: "베론 스틸블레이드"
  race: human
  faction: central_empire
  
  # 직위 (정치적 역할)
  title: 제국 검사장
  title_type: appointed   # appointed(임명) / hereditary(혈통) / elected(선출) / conquest(무력)

  # 등급 & 레벨 (개인 능력)
  grade: 소드마스터
  level: 87
  age_ticks: 18000        # ~50세 (360틱/년)
  
  # 특성
  specialization: [sword_mastery, tactical_command]
  personality:
    loyalty: 0.85         # 충성심
    ambition: 0.60        # 야망 (높으면 반란 가능성)
    wisdom: 0.70          # 지혜 (외교 판단)
    courage: 0.95         # 용기 (전투 참여 의지)
```

## 6. 레벨 vs 등급 vs 직위 요약 표

```
레벨 87  →  소드마스터 등급  →  검사장 직위   (강하지만 정치력 낮음)
레벨 44  →  이름 있는 전사  →  황제 직위     (약하지만 권위 최고)
레벨 58  →  오크로드 등급    →  오크로드 직위 (능력=직위 일치 파벌)
레벨 300 →  고위 마법사 등급 →  숲의 왕 직위  (엘프: 오래 살면 높은 등급=직위)
```
